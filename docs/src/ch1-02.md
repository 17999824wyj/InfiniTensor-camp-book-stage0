## VS Code 插件

在 Windows 下，比较推荐使用 VS Code 进行 Rust 开发，如果我没记错，在 VS-Code 里面直接装“rust-anaylzer”这个插件就行了，不用做其它麻烦的操作。

## 特殊说明

在下载 rust 工具时，**注意 rustc 版本号，至少 rust version 1.79.0 (129f3b996 2024-06-10)**。
